Ninz
Icônes

A beautiful open-source icon library with 7,101 icons across 6 distinct styles. Available as React components, CSS classes, or CDN — choose what fits your project.

✦ 7,101 icons ✦ React + TypeScript ✦ Tree-shakeable ✦ 6 styles
01

Icon Styles

Each style offers a unique visual character, giving you the flexibility to match any design language.

Bold 1,233 icons

Solid, high-impact icons with thick strokes and filled shapes. Bold style commands attention — perfect for primary actions, feature highlights, and anywhere you need icons that truly stand out.

solid fills thick strokes high contrast
Bold Duotone 1,077 icons

Bold strokes combined with duotone layering — a primary solid shape paired with a subtle secondary layer. Creates depth and dimension while maintaining the confident weight of bold style.

duotone depth layered fills dimensional
Broken 1,241 icons

Bold icons with intentional gaps and disconnected strokes. Adds visual intrigue and modern edge — great for tech brands, creative portfolios, and anywhere you want icons with personality.

intentional gaps disconnected modern edge
Line Duotone 1,085 icons

Thin strokes with two-tone depth — a primary thin outline with a secondary filled layer. Offers sophistication and subtle dimension. Perfect for cards, feature sections, and elegant UI.

light strokes two-tone sophisticated
Linear 1,233 icons

Clean, consistent single-weight strokes. Linear style is timeless and versatile — ideal for navigation, text-heavy interfaces, and anywhere icons need to be readable without overpowering.

single weight clean strokes versatile
Outline 1,235 icons

Minimalist hollow icons with open shapes and balanced proportions. The lightest touch in the collection — perfect for empty states, secondary actions, and minimalist designs.

hollow shapes minimalist open forms
02

React Installation

Install via your preferred package manager. The package includes full TypeScript support and is tree-shakeable for optimal bundle size.

npm 
npm install @ninzapp/solar-icons
yarn 
yarn add @ninzapp/solar-icons
pnpm 
pnpm add @ninzapp/solar-icons
✦ The package requires React 17+. Every icon is a named export — bundlers like Vite and webpack will only ship the icons you actually import.
03

React Usage

Named Import (Recommended)

Import icons directly by name for optimal tree-shaking. Only the icons you import will be included in your bundle.

Named Import 
import { Icon4kBold, IconHomeLinear, IconSettingsOutline } from '@ninzapp/solar-icons'

function App() {
  return (
    <div>
      <Icon4kBold size={32} />
      <IconHomeLinear size={24} color="gray" />
      <IconSettingsOutline size={20} style={{ color: 'royalblue' }} />
    </div>
  )
}

Dynamic Import

Use the generic Icon component for dynamic icon selection.

Dynamic Import 
import { Icon } from '@ninzapp/solar-icons'

function App() {
  return (
    <div>
      <Icon name="4k" variant="bold" size={32} />
      <Icon name="home" variant="linear" size={24} />
      <Icon name="settings" variant="outline" />
    </div>
  )
}

04

React API Reference

Icon Component (Dynamic)

Props
Prop Type Default Description
name string — Icon name in kebab-case (e.g. "add-circle")
variant string — Style variant: "bold" · "bold-duotone" · "broken" · "line-duotone" · "linear" · "outline"
size number | string — Sets both width and height. Takes precedence over width/height props.
width number | string — Icon width
height number | string — Icon height
color string currentColor Icon fill/stroke color
className string — CSS class for styling
style object — Inline styles
✦ All standard SVG props are also supported (e.g. onClick, aria-label, etc.)
05

CSS Method — Class-based

The simplest way for non-React projects. Add one stylesheet, then use <i> tags with classes anywhere in your HTML.

Add to your <head> 
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/gh/ShakilAnsary9/ninz-icones@v1.3.0/cdn/CSS/ninz-icons.css"/>
Use in HTML 
<!-- Format: ni  +  ni-{icon-name}-{variant}  +  ni-{size} -->

<i class="ni ni-alarm-bold ni-lg"></i>
<i class="ni ni-heart-linear ni-xl"></i>
<i class="ni ni-settings-outline ni-lg" style="color:#e2ff5d"></i>
<!-- With animation -->
<i class="ni ni-refresh-bold ni-spin ni-lg"></i>
<i class="ni ni-refresh-outline ni-spin-slow" style="--ni-speed:3s"></i>

06

Web Component — <ninz-icon>

Add one script tag and use <ninz-icon> anywhere. Injects real inline SVG — perfect for duotone, animations, and dynamic control.

Add to your <head> 
<script src="https://cdn.jsdelivr.net/gh/ShakilAnsary9/ninz-icones@v1.3.0/cdn/JS/ninz-icon.js"></script>
Use in HTML 
<ninz-icon name="alarm" variant="bold" size="32"></ninz-icon>
<ninz-icon name="heart" variant="linear" color="#e2ff5d"></ninz-icon>
<ninz-icon name="settings" variant="outline"></ninz-icon>
07

Tree-Shaking

Every icon is a named export — bundlers like Vite, webpack, and esbuild will only ship the icons you import.

Example 
// Only Icon4kBold is bundled
import { Icon4kBold } from '@ninzapp/solar-icons'

// IconHomeLinear is NOT included in your bundle
✦ Tree-shaking ensures your bundle only includes the icons you actually use — significantly reducing production bundle size.
