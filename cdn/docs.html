CDN Documentation

Sora
Icônes

A beautiful open-source icon library. Drop in a single line and get access to thousands of icons across 5 styles — no build tools required.

✦ 5 styles ⚡ jsDelivr CDN ◈ Web Component + CSS
01

CSS Method — Class-based

The simplest way. Add one stylesheet, then use <i> tags with classes anywhere in your HTML. No JavaScript required.

Step 1 — Add to your <head>
<!-- Sora Icônes CSS -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@latest/cdn/sora-icons.css"/>
<script src="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@latest/cdn/sora-icons-shim.js"></script>
Step 2 — Use in HTML
<!-- Format: si  +  si-{style}  +  si-{icon-name} -->

<i class="si si-outline si-alarm"></i>
<i class="si si-bold si-heart si-xl"></i>
<i class="si si-duotone si-calendar si-2xl" style="color:#e2ff5d"></i>
<i class="si si-linear si-bell si-spin"></i>  <!-- spinning -->
CSS Class Reference
ClassValuesDescription
si—Required base class on every icon
si-{style}bold · broken · duotone · linear · outlineIcon style variant
si-{name}alarm · heart · bell …Icon name
si-xs / sm / md / lg / xl / 2xl / 3xl0.75rem → 3remSize helper
si-spin—Continuous rotation animation
si-pulse—Step-based spin (loading feel)
si-muted / primary / success / warning / danger—Preset color helpers
✦ Control color with the CSS color property — either inline or via a CSS class. The icon inherits currentColor automatically.
02

Web Component — <sora-icon>

The power method. Add one script tag and use <sora-icon> anywhere. Injects real inline SVG — perfect for duotone, animations, and dynamic control.

Step 1 — Add to your <head>
<script src="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@latest/cdn/sora-icon.js"></script>
Step 2 — Use in HTML
<!-- Basic -->
<sora-icon name="alarm" style="outline"></sora-icon>

<!-- With size + color -->
<sora-icon name="heart" style="bold" size="32" color="#e2ff5d"></sora-icon>

<!-- Custom stroke width -->
<sora-icon name="bell" style="linear" size="24" stroke-width="1"></sora-icon>

<!-- Dynamic via JS -->
<sora-icon id="my-icon" name="moon" style="outline"></sora-icon>
<script>
  document.getElementById('my-icon').setAttribute('name', 'sun');
</script>
Attributes
AttributeDefaultDescription
name—Icon name (e.g. alarm). Required.
styleoutlineStyle variant: bold · broken · duotone · linear · outline
size1emWidth & height — any CSS unit or plain number (px)
colorcurrentColorOverrides stroke/fill. Inherits from CSS by default.
stroke-widthSVG defaultOverride the icon's stroke width
✦ The web component caches every fetched SVG in memory — repeat uses of the same icon cost zero extra network requests. Attributes are fully reactive; update them via JS and the icon re-renders instantly.
03

Version Pinning

Using @latest always pulls the newest release. For production, pin to a specific Git tag to avoid unexpected changes.

Pinned URL format
<!-- Latest (development) -->
https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@latest/cdn/sora-icons.css

<!-- Pinned to v1.0.0 (production recommended) -->
https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@1.0.0/cdn/sora-icons.css

<!-- Pinned to a specific commit hash -->
https://cdn.jsdelivr.net/gh/YOUR_USERNAME/sora-icones@a3f92c1/cdn/sora-icons.css
